# BHEL HR Management System

## Startup Procedure

To start the entire system locally for testing, simply run the included automated startup script:

1. Open a command prompt or terminal in the `Assignment/` folder.
2. Double-click **`start.bat`** (or run `.\start.bat` from your terminal).

This automated setup script will:
- Check if the project is compiled, and compile it automatically if needed.
- Start the Database Server (running on port `1099`) in a new window.
- Start the Application Server (running on port `1100`) in a new window.
- Launch the HR Client in a separate window.
- Launch the Employee Client in a separate window.

### Stopping the System
To stop the servers and clients, simply close their respective Command Prompt windows.
